Media Toolkit - Open-source media app for Windows 10/11
https://github.com/Keylem2/media_toolkit

INSTALL (most users):
  Download MediaToolkit-Setup.exe from the latest GitHub release.
  Run it — that's it. Standalone: no Python, FFmpeg, or other installs needed.
  Launch Media Toolkit from Start Menu or desktop.
